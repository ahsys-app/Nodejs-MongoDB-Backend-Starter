## Getting started
* Make sure you install **nodemon** globally `npm install -g nodemon`.
* Recommended node js 14.x (suggestion v14.9.0) and npm 6+. (suggestion v6.14.8).
* Install dependencies: npm install.

```
npm run serve
```
